```c
if(flags.flagV ){

```
Эта строка проверяет является ли ch управляющим символом.
```c
        if((ch < 32 && ch != '\n' && ch != '\t')|| ch == 127){
          printf("^");
        }
```

        if((ch < 32 || (ch > 126 && ch < 160)) && ch != '\n' && ch != '\t'){
